# The Deborah Network

> Empowering Women. Raising Leaders. Transforming Nations.
> _From Glory to Glory, until we see The King._

The official website of **The Deborah Network (TDN)** — a global community of women taking a stand for the Kingdom of God on earth, for this generation and the next. Inspired by the biblical Deborah, the platform raises purpose-driven women through faith, mentorship, leadership development, and community impact.

## Design

- **Style:** Modern · Elegant · Feminine · Premium · Kingdom-Focused
- **Palette:** Deep Plum `#4A234F`, Royal Gold `#D4AF37`, Soft Ivory `#FAF8F5`
- **Type:** Playfair Display / Cormorant Garamond (headings), Inter (body)

## Pages

`index` · `about` · `founder` · `programs` · `events` · `membership` · `resources` · `impact` · `partner` · `contact`

## Features

- Responsive design with mobile slide-out navigation
- Animated impact counters, testimonial carousels, and live event countdowns
- Resource hub with category filtering
- **Deborah AI Mentor** floating assistant widget
- Prayer Wall & Leadership Assessment logic
- Scroll-reveal animations and demo-handled forms

## Structure

```
.
├── index.html + 9 inner pages
├── assets/
│   ├── css/styles.css   # design system
│   ├── js/main.js       # interactions
│   └── images/          # photography
└── README.md
```

## Running locally

It's a static site — open `index.html` in a browser, or serve the folder:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```
